# FateTools

FateTools is a quality-of-life enhancement module for the Fate Core Official system on Foundry VTT.

The goal of FateTools is to bring important Fate mechanics directly onto the canvas, reducing the need to open character sheets and utility windows during play.

Requires:
- Foundry VTT v14
- Fate Core Official v4.6.17.12+

# Features

## Active Aspects

Provides a centralized list of:

- Game Aspects
- Scene Aspects
- Zone Aspects
- Character Aspects

Supports:

- Free Invoke tracking
- Invoke spending
- GM Fate Point spending
- Player Fate Point spending

## Token Overlays

Hover or select a token to view:

- Stress tracks
- Consequences

Stress boxes can be modified directly from the canvas.

## Scene HUD

Displays:

- Game Aspects
- Scene Aspects
- Countdown Tracks

Supports:

- Aspect creation
- Countdown interaction
- Free Invoke tracking

## Zones

Create and manage Fate Zones directly on the canvas.

Supports:

- Zone creation
- Zone Aspects
- Zone stress tracks
